# vpn-guard

Local Clash Verge Rev guardian for macOS.

This repository archives the implementation notes and support files for a local
guardian that keeps Clash Verge Rev in TUN + rule mode, refreshes the active
remote subscription on a 7-day schedule, and switches to the fastest Japan node
when the current node becomes slow or unhealthy.

Main document: [`vpn-guard.md`](./vpn-guard.md)

No subscription URL, node password, token, private key, or raw Clash backup file
is intentionally included in this public archive.
